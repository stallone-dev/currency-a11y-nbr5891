# Mapeamento de Recomendações eMAG 3.1

O eMAG 3.1 divide as recomendações em 6 áreas principais. Use este mapa para auditorias rápidas.

## 1. Marcadores (Markup)
- **1.1:** HTML Válido.
- **1.2:** Estrutura Semântica.
- **1.3:** Títulos (`h1` a `h6`) lógicos.
- **1.4:** Separar conteúdo de apresentação (CSS).
- **1.5:** Alternativa para Scripts (ex: `<noscript>`).

## 2. Comportamento (Behavior)
- **2.1:** Navegação lógica por teclado.
- **2.2:** Evitar "Keyboard Traps".
- **2.3:** Controle de tempo (`timing-adjustable`).
- **2.4:** Prevenir erros (erros claros, submissão confirmada).

## 3. Conteúdo / Informação (Content)
- **3.1:** Alternativa textual para imagens.
- **3.2:** Alternativa para áudio/vídeo (legendas, audiodescrição).
- **3.3:** Linguagem Simples e clara.
- **3.4:** Identificar o idioma da página.
- **3.5:** Títulos de página descritivos.

## 4. Apresentação / Design (Presentation)
- **4.1:** Contraste de cor.
- **4.2:** Não usar apenas cor para passar informação.
- **4.3:** Redimensionamento de texto (zoom 200%).
- **4.4:** Indicador de foco visível.

## 5. Formulários (Forms)
- **5.1:** Rótulos (`label`) para todos os campos.
- **5.2:** Agrupamento de campos (`fieldset`).
- **5.3:** Mensagens de erro em texto e associadas.

## 6. Navegação (Navigation)
- **6.1:** Links para saltar conteúdo (skip links).
- **6.2:** Links com texto descritivo.
- **6.3:** Menu de navegação consistente.
- **6.4:** Barra de Acessibilidade (Padrão GovBR).
- **6.5:** Integração VLibras.
